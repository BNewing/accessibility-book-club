{{releaseVersion | log }}

If you would love to contribute a discussion guide, we would love to see it!

There is a template for a discussion guide, which I recommend you use when formulating questions. There are two ways of submitting a discussion guide:

  • Submit a PR to the 'Accessibility Book Club' repo
  • Fill out 'Accessibility Book Club' Discussion Guide submission form

This is run on a voluntary basis, so contributions will be looked at as and when I can - but very keen to see submissions, and grow this as a resource. You will be attributed as the creator of the discussion guide.
